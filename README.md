# Portfolio
This is my personal portfolio website created to showcase my skills, projects, and technical journey as a Computer Science Engineering graduate and aspiring software developer.

The portfolio includes a modern responsive design, dark mode toggle, typing animation, and smooth scroll reveal effects to create a professional user experience.

Features:
- Responsive modern UI
- Dark / Light mode
- Typing animation
- Scroll reveal animations
- Skills and project showcase
- Contact section with social links

Technologies Used:
HTML, CSS, JavaScript

This project represents my continuous learning in web development and my passion for building real-world applications.
